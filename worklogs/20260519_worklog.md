Target
  Prove the reverse direction as certified trace synthesis:

  theorem activeWitness_of_not_dSeparated
      (hnot : ¬ DAG.dSeparated G X Y Z) :
      ActiveWitness G X Y Z

  then derive:

  theorem dSeparated_of_dSeparates
      (hsep : dSeparates G X Y Z) :
      DAG.dSeparated G X Y Z

  and finally package the disjoint-domain equivalence:

  theorem dSeparates_iff_dSeparated_disjoint
      (hXYZ : DisjointSets X Y Z) :
      dSeparates G X Y Z ↔ DAG.dSeparated G X Y Z

  The key choice: the reverse theorem should produce an ActiveWitness,
  not necessarily preserve the exact original endpoints x and y. That
  makes rerouting through leaked ancestry to another node in X or Y legal
  and much easier.

  Phase 1: Add Explicit Static Path IR
  Do not work directly against SimpleGraph.Reachable or MAGWalk.trans at
  first; they hide too much structure.

  Introduce a Type-valued path for static graph evidence, probably in a
  new file like DSeparation/TraceSynthesis.lean:

  inductive StaticStep (G : DAG) (X Y Z : Finset ℕ) : ℕ → ℕ → Type
    | direct ...
    | moralJump ...

  inductive StaticRoute (G : DAG) (X Y Z : Finset ℕ) : ℕ → ℕ → Type
    | nil ...
    | cons ...

  StaticStep.direct mirrors a surviving directed/reversed DAG edge.
  StaticStep.moralJump stores the common child w for a co-parent edge u
  -> w <- v.

  Needed bridge lemmas:

  StaticStep.toMAGWalk
  StaticRoute.toMAGWalk
  nonemptyStaticRoute_of_dSeparationGraph_reachable

  The last one can be noncomputable at first. The important part is
  preserving the step witnesses.

  Phase 2: Expand Static Routes Into Raw Trace Skeletons
  Define an expanded trace representation that records oriented DAG
  traversals.

  A direct static step emits one DAG traversal.
  A moral jump emits two traversals:

  u -> w <- v

  This expansion must preserve enough direction information to know
  whether each junction is a collider or non-collider.

  The object should be closer to BayesBallPathT than to Trail, because
  the junction obligations are directional:

  def RawTrace.toStates : ...
  def RawTrace.LocalOpen : Prop

  or directly:

  inductive OpenTrace ...

  where each extension carries:

  ¬ DirectionalTripleBlocked G Z current arrival departure

  Phase 3: Prove Open Trace → ActiveRoute
  This is Lemma 2.

  Once local obligations are attached to each step, synthesis should be
  mechanical:

  def activeRoute_of_openTrace :
    OpenTrace G Z s t → ActiveRoute G Z s t

  Then existing code closes the semantic witness:

  ActiveRoute.to_activeTrail :
    ActiveRoute G Z s t →
    ∃ tr : Trail G s.1 t.1, ¬ tr.isBlocked Z

  This phase should be done before path normalization, because it gives a
  clean target type for the hard lemma.

  Phase 4: Path Normalization / Rerouting
  This is Lemma 1 and the hard part.

  Define a bad collider in the expanded route:

  BadCollider := collider window ∧ Disjoint ({w} ∪ descendants G w) Z

  A normalized route has no bad colliders:

  def NormalizedStaticRoute ... : Prop :=
    ∀ collider window, ¬ Disjoint ({w} ∪ descendants G w) Z

  Prove:

  theorem normalize_staticRoute :
    StaticRoute G X Y Z x y →
    x ∈ X →
    y ∈ Y →
    Σ x', x' ∈ X ∧ Σ y', y' ∈ Y ∧
      NormalizedStaticRoute G X Y Z x' y'

  The endpoint may change. That is the key design decision.

  Rerouting argument:

  - If bad collider child w is ancestral to Z, contradiction: then it is
    Z-open.
  - Therefore its ancestral target lies in X or Y.
  - If it leaks to X, replace the prefix by the descendant route to that
    X endpoint.
  - If it leaks to Y, replace the suffix similarly.
  - Use a minimality measure, probably route length plus number of bad
    colliders, to make Lean accept termination.

  Do this over StaticRoute, not MAGWalk.

  Phase 5: Normalized Route → Open Trace
  Prove the compiler theorem:

  theorem openTrace_of_normalized_staticRoute :
    NormalizedStaticRoute G X Y Z x y →
    Σ d, OpenTrace G Z (x, TrailDir.outOf) (y, d)

  Local cases:

  - Non-collider junction: current node is a surviving static node, hence
    current ∉ Z, so it is open.
  - Collider junction: normalization supplies descendant intersection
    with Z, so it is open.
  - Moral jump child may be deleted from the static graph, but it is
    allowed inside the trace as a skipped collider witness.

  Phase 6: Final Assembly
  From ¬ DAG.dSeparated, get:

  ∃ x ∈ X, ∃ y ∈ Y,
    (G.dSeparationGraph X Y Z).Reachable x y

  Then:

  Reachable
  → StaticRoute
  → NormalizedStaticRoute
  → OpenTrace
  → ActiveRoute
  → ∃ Trail, ¬ isBlocked
  → ¬ dSeparates

  Final theorem:

  theorem dSeparated_of_dSeparates :
    dSeparates G X Y Z → DAG.dSeparated G X Y Z

  Then combine with existing:

  dSeparated_of_dSeparated_disjoint :
    DisjointSets X Y Z →
    DAG.dSeparated G X Y Z → dSeparates G X Y Z

  to get the iff.

  Order Of Work

  1. Add StaticStep / StaticRoute.

  3. Add OpenTrace and prove OpenTrace → ActiveRoute.
  4. Prove direct-edge and open-collider step compilers.
  5. Prove normalization.
  6. Assemble ActiveWitness.
  7. Package final iff and run lake build.

  The highest-risk item is Phase 4. Everything else is engineering around
