```lean
inductive StaticStep (G : DAG) (X Y Z : Finset ℕ) : ℕ → ℕ → Type where

inductive StaticRoute (G : DAG) (X Y Z : Finset ℕ) : ℕ → ℕ → Type where
```

## Phase 2: Open Trace IR ✅ COMPLETE

The `OpenTrace` inductive type lives in `DSeparation/TraceSynthesis/OpenTrace.lean`:

```lean
inductive OpenTrace (G : DAG) (Z : Finset ℕ) : ℕ × TrailDir → ℕ × TrailDir → Type where
  | nil (s : ℕ × TrailDir) : OpenTrace G Z s s
  | cons {u v w : ℕ} {arrival departure finalDir : TrailDir}
      (hEdge : TrailDir.edgeIntoCurrent G u v departure)
      (hOpen : ¬ DirectionalTripleBlocked G Z u arrival departure)
      (rest : OpenTrace G Z (v, departure) (w, finalDir)) :
      OpenTrace G Z (u, arrival) (w, finalDir)
```

Each `cons` node carries a **local non-blocking proof**, making `OpenTrace` a `Type`-valued certificate that every junction (collider or non-collider) is `Z`-open. This is the *decompiler's* target IR: it records oriented DAG traversals with enough direction information to know whether each junction is a collider or non-collider.

### Rerouting Argument Sketch (formalized)

1. If bad collider child `w` is ancestral to `Z`, contradiction: then it is `Z`-open.
2. Therefore its ancestral target lies in `X` or `Y` (via `ancestor_escape` in `TraceSynthesis/Graph.lean`).
3. If it leaks to `X`, replace the prefix by the descendant route to that `X` endpoint.
4. If it leaks to `Y`, replace the suffix similarly.
5. Use bad-collider count minimality to force a zero-bad witness.

> **Constraint:** Do this over `StaticRoute`, not `MAGWalk`.

## Phase 6: Final Assembly ✅ COMPLETE

From `¬ DAG.dSeparated`, the full pipeline is wired in `TraceSynthesis/Assembly.lean`:

```lean
theorem activeWitness_of_not_dSeparated {G : DAG} {X Y Z : Finset ℕ}
    (hnot : ¬ DAG.dSeparated G X Y Z) :
    ActiveWitness G X Y Z := by
  unfold DAG.dSeparated at hnot
  push Not at hnot
  rcases hnot with ⟨x, hx, y, hy, hreach⟩
  have ⟨route⟩ := nonemptyStaticRoute_of_dSeparationGraph_reachable hreach
  have hwit : ∃ w : StaticRouteWitness G X Y Z, True :=
    ⟨⟨x, hx, y, hy, route⟩, trivial⟩
  rcases normalized_route_exists_of_improves hwit route_improves_of_bad with
    ⟨wmin, hzero⟩
  rcases activeRoute_of_countBadColliders_zero hzero with
    ⟨⟨finalDir, activeRoute⟩⟩
  exact ⟨wmin.x, wmin.hx, wmin.y, wmin.hy, finalDir, ⟨activeRoute⟩⟩
```

This is a complete proof. The chain is:

```
Reachable
  → StaticRoute            (TraceSynthesis/StaticRoute)
  → NormalizedStaticRoute  (TraceSynthesis/MinimalWitness + Assembly)
  → OpenTrace              (TraceSynthesis/OpenTrace)
  → ActiveRoute            (TraceSynthesis/OpenTrace)
  → ∃ Trail, ¬ isBlocked   (ActiveRoute.to_activeTrail)
  → ¬ dSeparates
```

The reverse direction feeds the final equivalence:

```lean

  dSeparates G X Y Z → DAG.dSeparated G X Y Z
```

Then combine with the existing forward theorem:
